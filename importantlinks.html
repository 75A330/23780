Remove Write Protection Using Diskpart (Windows) If your SD card is still write-protected and there is no physical switch, you can use the diskpart command in Windows to try and remove the write protection. Steps: Open Command Prompt as Administrator: Press Windows + X and select Command Prompt (Admin) or Windows PowerShell (Admin). Launch Diskpart: In the Command Prompt window, type diskpart and press Enter. List all disks: Type list disk and press Enter to see all the disks connected to your computer. Select your SD card: Type select disk X (replace X with the number of your SD card) and press Enter. Clear the write protection: Type attributes disk clear readonly and press Enter. This command should remove the write protection from the SD card. Exit Diskpart: Type exit and press Enter to close Diskpart.
